# @amlplugins/xai-images

Native `@ai-sdk/xai` SDK re-exported under the `@amlplugins` namespace.

> xAI Image Generation — Grok image models (grok-2-image, grok-2-image-1212) via @ai-sdk/xai imageModel adapter.

## Install

```bash
npm install @amlplugins/xai-images
```

## Use

```ts
import { /* named exports from @ai-sdk/xai */ } from "@amlplugins/xai-images";
// or
import sdk from "@amlplugins/xai-images";
```

This package additionally re-exports `ai` under the `secondary` namespace.

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
